# Проект: Обучение с учителем: качество модели

## Заказчик исследования: 
Интернет-магазин «В один клик» продаёт разные товары: для детей, для дома, мелкую бытовую технику, косметику и даже продукты.

## Источник данных:
Данные компании по постоянным клиентам.

# Задачи для исследования:
1. Промаркировать уровень финансовой активности постоянных покупателей. В компании принято выделять два уровня активности: «снизилась», если клиент стал покупать меньше товаров, и «прежний уровень».
2. Собрать данные по клиентам по следующим группам: 
* Признаки, которые описывают коммуникацию сотрудников компании с клиентом.
* Признаки, которые описывают продуктовое поведение покупателя. Например, какие товары покупает и как часто.
* Признаки, которые описывают покупательское поведение клиента. Например, сколько тратил в магазине.
* Признаки, которые описывают поведение покупателя на сайте. Например, как много страниц просматривает и сколько времени проводит на сайте.
3. Построить модель, которая предскажет вероятность снижения покупательской активности клиента в следующие три месяца.
4. В исследование нужно включить дополнительные данные финансового департамента о прибыльности клиента: какой доход каждый покупатель приносил компании за последние три месяца.
5. Используя данные модели и данные о прибыльности клиентов, нужно выделить сегменты покупателей и разработать для них персонализированные предложения.

При помощи пайплайнов были обучены 4 модели (DecisionTreeClassifier, KNeighborsClassifier, LogisticRegression и SVC) для предсказания вероятности снижения активности пользователей и найдена та, что показывает наилучшие результаты. Ею оказалась модель логистической регрессии со следующими параметрами: C=3, penalty='l1', random_state=42, solver='liblinear'

В итоге были изучены пользователи на основе предсказаний лучшей выбранной модели и получаемой от них прибыли.
В результате был выделен наиболее важный для изучения на данный момент сегмент: пользователи, приносящие высокую прибыль, но близкие к уходу. Такой сегмент был выделен приоритетным, так как большая часть пользователей сейчас все еще более лояльна, но многие начали уходить, и, так как компании важна прибыль, наиболее весомы потери клиентов, которые приносят больше прибыли. Поэтому нам очень важно большую часть из них удержать.
